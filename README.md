## What I learned

- flexbox:
```html
    <div id="main">...</div>
```
```css
    #main{
        display: flex;
        flex-flow: column nowrap;
    }
```

- hover Pseudo Class:
```html
    <div id="button">...</div>
```
```css
    #button:hover{
        text-decoration: underline;
    }
```

- title attribute:
```html
    <a id="" href="" target="blank" title="wikipedia" decoration="none" >Read more</a>
```
